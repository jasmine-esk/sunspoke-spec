# Sunspoke — Master Spec

Private working spec for Sunspoke v0 (Plaud-rework era → custom hardware).

The spec is published as a browsable page at GitHub Pages. Comments and proposed edits flow through GitHub Discussions and are periodically integrated into the main spec.

## Files

- `sunspoke-master-spec.md` — Canonical master spec (source of truth)
- `index.html` — Browsable web view with sidebar navigation and inline commenting

## How to view

Visit the GitHub Pages URL (listed under the repo's About section).

## How to comment

- Inline per-section comments: scroll to the bottom of any page and use the comment widget (GitHub login required to post, no login required to read)
- Threaded discussion: use the Discussions tab of this repo

## Updates

This repo is actively maintained. New comments are reviewed periodically and integrated into the spec. Change history is preserved in commit history.
